<p>🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS
- JavaScript
- Git e Github
- Figma

💻 Projeto

O DevLinks é um agregador de Links para utilizar como cartão de visitas online. Foi utilizado a adaptação para desktop e mobile.
